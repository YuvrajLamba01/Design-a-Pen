Transitions:
  • CLOSED  → OPEN    : start() + hasInk()
  • OPEN    → IN_USE  : write() + hasInk()
  • IN_USE  → OPEN    : after write
  • OPEN    → CLOSED  : close()
  • IN_USE  → CLOSED  : close()
  • (any)   → EMPTY   : no ink available
  • EMPTY   → CLOSED  : refill()
  • CLOSED  → CLOSED  : refill()
